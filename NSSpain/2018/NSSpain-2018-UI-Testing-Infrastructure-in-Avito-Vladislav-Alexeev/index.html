UI Testing Infrastructure in Avito - Vladislav Alexeev

We rely on UI testing heavily: we now have more than 500 e2e tests that we need to run and maintain. The perfect infrastructure is a key of the success. I cover what we use for testing, what problems we’ve met and how we solve them.
